Staves

Staves are tools that can be used to achieve different effects! Staves are crafted using a Base Staff, found by interacting with the Anchor in the Spires. The general recipe is as follows: where X is a type of flower

  • Jump Staff: Upon using on a block, will launch you upwards (negating all falldamage)
  • Ice Staff: Upon using on a block, will create a platform of ice
  • Quake Staff: Has a farther reach which upon using on a block, will cause an explosion (the explosion deals no damage to you)
  • Cloud Staff: Will either place a soft cloud beneath the player to cushion a fall, or on a used block
  • Inferno Staff: Has a farther reach which upon using on a block, will cause the area around it to be lit ablaze

  • Bloom Staff: Upon using on dirt/grass, will grow a variety of flowers
  • Shroomspawn Staff: Upon using on dirt/grass, will grow a variety of flowers

World Generation

Chronological Chronicles changes and or adds a variety of structures and world generation in vanilla Minecraft.

Spires

Spires are large structures found sporadically across the world. They are composed of gravel and Shingle Gravel, as well as having an Anchor inside. In front of the spire is a sign that denotes its name. Sometimes spires can be prime, meaning they will generate perfectly. They are a good checkpoint or hiding place in the night!

Marshy Seaside

The Marshy Seaside generates along the coast of the world and is composed of dirt, dried trees/leaves, and not much else

Dunes/Salt Flats

Dunes/Salt Flats will generate sporadically throughout the world

Snowy Mountaintops

Snowy Mountaintops will generate at the tops of mountains

Dungeons

Dungeons have been changed to now generate by default above ground (most of the time). They can spawn sometimes with multiple levels. Also, breaking a mob spawner is the only way to get the loot!

Gameplay

Cards

Cards are a new equippable series of items that provide passive effects to the user. They can be found/crafted in a variety of ways. They are subdivided into 4 suits, and 4 ranks

  • Hearts: Provide vitality related effects
  • Spades: Provide mining related effects
  • Diamonds: Provide building related effects
  • Clovers: Provide agility related effects

And the ranks:

  • Aces/Jacks: First type
  • King/Queen: Second type

Mob Sieges

Occasionally mobs can decide to siege a base, so beware

Artisanship

Since 1.2, a new gameplay mechanic to process and create new items/blocks has been added: artisanship. The first necessary components are the: Processor and the Age Rack, from which all artisan items are crafted. Below is a complete list of how to produce any of the new goods

  • [Processor] + (Apple) + (Empty Bottle) => Apple Extract
  • [Aging Rack] + (Apple Extract) => Apple Cider
  • [Processor] + (Wheat) + (Bowl) => Wheat Flour
  • [Processor] + (Reed) + (Bowl) => Sugar
  • [Crafting Table] + (Filled Bottle) + (Yeast) + (Wheat Flour) + (Sugar) => Wort
  • [Aging Rack] + (Wort) => Beer
  • [Water] + (Empty Bottle) => Filled Bottle
  • [Processor] + (Salt Block) + (Bowl) => Salt Pile
  • [Crafting Table] + (Yeast) + (Wheat Flour) + (Egg) + (Salt Pile) => Dough
  • [Furnace] + (Dough) => Bread
  • [Furnace] + (Filled Bottle) + (Wheat Flour) => Yeast
  • [Processor] + (Poison Mushroom) + (Apple Cider) => Vinegar
  • [Furnace] + (Any Mod Plant) + (Apple Cider) + (FUEL) => Spiced Cider
  • [Furnace] + (Any Mod Plant) + (Filled Bottle) + (FUEL) => Tea
  • [Aging Rack] + (Bottle Of Milk) => Curd
  • [Crafting Table] + (Bottle Of Milk) + (Vinegar) => Curd
  • [Aging Rack] + (Curd) => Moist Cheese
  • [Furnace] + (Filled Bottle) + (Curd) + (FUEL) => Mozzarella
  • [Processor] + (Milk) + (Bowl) => Butter
  • [Crafting Table] + (Butter) + (Wheat FLour) + (Sugar) + (Salt) => Pastry Dough
  • [Furnace] + (Filled Bottle) + (Pastry Dough) => Pastry
  • [Processor] + (Curd) + (Bowl) => Pressed Curd
  • [Furnace] + (Pressed Curd) + (FUEL) => Dry Cheese
  • [Aging Rack] + (Pressed Curd) => Hard Cheese

Hub

The hub is a new world added during the 1.2 update, which will force the game to load into a hub world, from which, other worlds can be travelled to.

In order to make your own hub you must follow these steps:

  1. Create a new world
  2. Cut and paste the original hub world directory, and place it somewhere safe
  3. Rename the world you created to 'Hub' (the directory that contains the world data)

And now your game should automatically load into THAT world everytime you log on. In order to actually make it be able to teleport you to other worlds you will need to do the following:

  1. Create a new .txt file in the directory of the minecraft world, named 'hub'
  2. Inside of the hub.txt file, paste the following as a template:

    3. canSaveGame:true

    canBreakBlocks:true

    World1:x:-160:y:46:z:-226

    World2:x:-172:y:46:z:-229

    World3:x:-181:y:46:z:-242

    World4:x:-145:y:46:z:-225

    World5:x:-155:y:46:z:-241

  3. Ensure that each is on its OWN seperate line, AND that there are no empty lines in between. You can add as many worlds (as well as worlds named something besides 'WorldX')
  4. Save the file, and next time you load into the hub, the settings should be applied

The syntax 'World1:x:-160:y:46:z:-226' defines a teleporter/loader/i don't have a good terminology picked out yet, with the first string seperated by a colon representing the world name, and the next x, y, and z, obviously the coordinates WHERE the teleport is to occur. The best way to find the coordinates for a teleporter is by using the F3 info menu

When editing your hub world, ensure that 'canSaveGame' is set to 'true'. While it is set to true, do NOT attempt to use the teleporters to the other worlds (because it will save the hubworld IN the exact space where the teleporter position would be, causing a softlock)

Once you're done with the construction of your world, make sure to set 'canSaveGame' and 'canBreakBlocks' to false.

Misc

  • Hoes can be right-clicked on fully grown wheat to harvest them, and automatically plant back the seed
  • Signs can now be edited
  • Torches will start snuffed out, but can be relit by simply pressing on them (mobs can also now snuff them out)
  • Most mobs can see through walls and will make their way towards the player
  • There exists a days system, where mobs will begin to increase in waves as the nights go on
  • The Numpad can be used to cheat in items/blocks (if you know the ID of the item)
